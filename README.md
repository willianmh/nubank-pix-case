# nubank-pix-case

This repository contains the resolution of the "Nubank Analytics Engineer Case".

It was developed with MySQL v8.0.23.

## Get started

First of all, let's create our database

~~~ sql
CREATE DATABASE nubank;
~~~

You can now run the commands on "nu-warehouse.sql" to create the tables.

Then you can run the commands on "nu-warehouse-populate.sql" to populate the tables.
